# Working Faster with Tabular Editor Hotkeys

> **Time to learn**: ~15 minutes | **Time saved**: 5-15 minutes per week

## The Problem

You're using Tabular Editor through menus and right-click actions. It works, but it's slow — especially for actions you repeat dozens of times per session like switching between objects, running BPA, or formatting DAX.

## The Solution

Learn the essential keyboard shortcuts that eliminate the most common mouse interactions. Even memorizing 5-10 hotkeys can dramatically speed up your workflow.

## Key Takeaways

- **Ctrl+F** / **Ctrl+H** — Find and Replace in TOM Explorer
- **F12** — Go to Definition
- **Ctrl+Shift+F** — Format DAX
- **Alt+1** — New Measure
- **Ctrl+Z** / **Ctrl+Y** — Undo / Redo
- Customize shortcuts via Tools → Preferences → Keyboard
