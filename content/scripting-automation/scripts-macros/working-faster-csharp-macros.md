# Working Faster with C# Macros

> **Time to learn**: ~15 minutes | **Time saved**: 5-15 minutes per week

## The Problem

You've written useful C# scripts for common tasks — bulk formatting, creating measure tables, fixing BPA violations. But each time you need one, you have to find the script file, open it, and run it manually.

## The Solution

Macros let you save C# scripts as reusable toolbar actions with names, icons, and keyboard shortcuts. Instead of finding and pasting a script, you click a button or press a hotkey.

## Key Takeaways

- Macros are saved C# scripts with a name, icon, and optional shortcut
- Access macros from the toolbar, right-click context menu, or keyboard shortcut
- Organize macros into folders for different workflows (BPA fixes, formatting, etc.)
- Share macros with your team by exporting the macro file
